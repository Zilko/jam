# Jam

Mod [commissioned](https://www.fiverr.com/zilkowo/make-you-a-geometry-dash-mod-of-your-design) by **carexistent**

<cp>Jam</c> has arrived to Geometry Dash, and with that, a new world of possibilities!

This mod adds a Jam currency to your game which can be used to buy things that you might or might not be able to unlock normally in the vanilla game!

# info

whenever you buy something it is only unlocked while you have the mod installed so ur savefile is not modified

jam data is saved locally so deleting the mod will possibly delete your jam
